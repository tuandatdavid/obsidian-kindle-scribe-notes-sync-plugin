# Obsidian Kindle Scribe Notes Sync Plugin

Sync your Kindle Scribe notes to your Obsidian Vault.

> [!WARNING]  
> This plugin is in *very, very early state*. I honestly barely have time to develop it, so contributions are welcome.

Current functionality:
- [x] pulling notes from root folder 
- [x] pulling notes from subfolders and recreating that structure in Obsidian Vault.
- [x] notes OCR
    - [x] first iteration will be just using OpenRouter API

Roadmap:
- [ ] separate out OCR and download flow
- [ ] autocheck notes hashes
